# Woodridge New York Trip App

A live trip web app for the 321 Villanova to Woodridge, NY trip.

## Website

https://venkata7364.github.io/woodridge-trip/

## Features

- Sunday trip timeline
- Important fixed places
- Nearby place search
- Google Maps links
- Shared photo upload/gallery
- Shared expense tracker
- Expense split per person
- Mobile-friendly design

## Trip Route

Starting point: 321 Villanova Rd, Glassboro, NJ  
Airbnb: 170 Silver Lake Rd, Woodridge, NY 12789

## Main Stops

- Hamilton Point and Castle Point Loop
- Wine tasting near Marlboro, NY
- Beacon Ferry Terminal
- NY Waterway Ferry area
- Indian restaurant options
- Extra nearby places and backups
